## Why use this toolkit

Building hackathon projects means working under tight deadlines. This toolkit helps you:

- **Skip the setup phase** - Jump straight into building with pre-tested integrations
- **Avoid common pitfalls** - Learn from real-world experience with each tool
- **Maximize free tiers** - Know exactly what credits and limits you have available
- **Ship production-quality work** - Use the same tools companies deploy in production

<Tip>
  Start with the [Quick Reference](/quick-reference) guide to grab free credits and copy-paste starters before your hackathon begins.
</Tip>

## Popular resources

### Frontend polish

- **[React Bits](https://www.reactbits.dev/)** - 110+ animated React components with minimal dependencies
- **[Aceternity UI](https://ui.aceternity.com/)** - 200+ free copy-paste components built with Tailwind CSS and Framer Motion

### AI integration

- **[Hugging Face](https://huggingface.co/)** - 45,000+ models from leading AI providers through unified API
- **[Langflow](https://www.langflow.org/)** - No-code AI agent and RAG app builder with visual drag-and-drop interface

### Voice agents

- **[LiveKit](https://livekit.io/)** - Open-source real-time voice AI framework with complete STT-LLM-TTS pipeline
- **[Twilio](https://www.twilio.com/en-us)** - Calling, SMS, and OTP integration with $15 free credits

### Free credits

- **[Vultr](https://www.vultr.com/promo/try250)** - $250-300 free cloud credits for 30 days
- **[Serper.dev](https://serper.dev/)** - 2,500+ free Google Search API queries

<Warning>
  Many services require verification or signup time. Create your accounts before the hackathon starts to avoid delays.
</Warning>

## How to contribute

Found a great tool that helped you win a hackathon? Contributions are welcome!

**What to include:**
- Tool name and official link
- What it does (1-2 sentences)
- Why it's valuable for hackathons
- Free tier details or pricing
- Personal experience if applicable

**Focus areas:**
- India-specific tools (e.g., Razorpay test mode)
- Emerging AI platforms
- Developer tools with generous free tiers
- Real-time/WebRTC technologies
- Data processing and ETL tools
